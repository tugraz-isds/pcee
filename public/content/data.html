Multidimensional Data

Multidimensional data refers to data that is organised across multiple dimensions, allowing it to be analysed from various perspectives. Each dimension represents a different attribute or feature of the data. Each record has a data value on each dimension.

Analysing multidimensional data enables:

  • Identifying patterns and outliers.
  • Recognising relationships between different attributes.
  • Uncovering trends.

Visualising multidimensional data through techniques like parallel coordinates plots, scatter plots, or heatmaps helps to gain a clearer understanding of how different variables interact and influence each other. Each of these techniques highlights different aspects of the data, and when combined, they provide a more comprehensive analysis:

  • Parallel coordinates plots:
    • Ideal for visualising relationships across multiple dimensions simultaneously.
    • Especially useful for comparing different records across several dimensions.
  • Scatter plots:
    • Effective for identifying correlations between two variables.
    • Great for spotting trends and outliers.
  • Heatmaps:
    • Provide an intuitive way to visualise the density and intensity of relationships between multiple variables.
    • Useful for identifying patterns that may not be immediately obvious.

How these techniques work together:

  • When these visualisations are used together, they offer a more versatile analysis of the data.
  • Each technique highlights different aspects of the data, helping to understand how variables interact with each other.
  • By cross-referencing insights from multiple visualisations, a more comprehensive view of the dataset can be obtained.
