3. Operations on Records

Interactivity brings a parallel coordinates chart to life. Some interactive operations relate to records:

  • Filtering Records

    Records can be activated and deactivated by setting filters on one or more dimensions. Records outside of the range are automatically deactivated and greyed out.

    Use the double-edged range slidersto filter out some records. Filters can also be set by right-clicking on a dimension's name to open its context menu.

  • Selecting Records

    It is helpful to be able to select one or more records in the dataset and to highlight them accordingly.

    Left-click one or more records to highlight them in orange. A shift-left-click adds one or more records to the current selection. A control-left-click toggles the selection status of the affected records. Left-clicking in empty space clears the current selection.

  • Hovering over Records

    If there are a large number of records, a parallel coordinates visualisation contains a large number of potentially crossing and overlapping polylines, so it is challenging to recognise which polyline belongs to which record.

    Hover over one or more records to highlight the record(s) beneath the mouse pointer in red, and to show tooltips with the values of the corresponding record(s).

  • Other Operations via Record Context Menu

    When hovering over one or more records, right-clicking opens the Record Context Menu with the following operations: Select Record(s), Unselect Record(s), Toggle Record(s), Add to Selection, and Remove from Selection.

    Hover over a record and right-click to open the Record Context Menu.
