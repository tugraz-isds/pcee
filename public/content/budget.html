The Household Finances Dataset

The houshold finances dataset was created to get familiar with parallel coordinates and to experiment with them. In the table below, you can add or remove columns and rows as well as modify the values. It also allows you to explore the interactivity in the chart described in the section “Interactive Data Exploration”, helping you practice how to work with the chart interactively.

The dataset represents household finances with 6 sample records. Each row corresponds to one person and contains five dimensions:

  • Name: Identifier for each person.
  • Income: Monthly household income.
  • Fixed Costs: Monthly fixed expenses.
  • Savings: Monthly amount saved.
  • Debt: Outstanding debt balance.
