Interactive Data Exploration

A static parallel coordinates plot provides an overview of the entire dataset, but only interactivity really brings it to life. Numerous interaction techniques help an analyst explore a multidimensional dataset and find relevant insights. Some interaction techniques relate to dimensions (variables) of the data: showing and hiding dimensions, adjusting a dimension’s range, moving dimensions, and inverting dimensions, as well as displaying histograms. Others relate to records (data points): selecting, brushing and linking, filtering, and hovering over records.

Inverting Dimensions

By default, a dimension’s axis places higher values at the top and lower values at the bottom. Inverting a dimension can be helpful to corroborate a suspected correlation between adjacent dimensions:

  • Clicking the arrow above the dimension axis.
  • Right-clicking the dimension name to activate the context menu and selecting Invert.

Moving Dimensions

When looking for correlations between dimensions, meaningful relationships are only revealed between adjacent dimensions. Hence, it is important to be able to move one dimension next to another dimension of interest.

Option to Move a Dimension in the Chart:

With drag and drop a dimension can be moved at the desired position.

Hiding Dimensions

Displaying too many dimensions can be overwhelming, so it is important to be able to hide individual dimensions.

Option to Hide a Dimension in the Chart:

  • A dimension can hidden by right-clicking the dimension name to activate the context menu and selecting Hide.
  • Hidden dimensions can be shown from every dimensions context menu, by clicking on Show All.

Adjusting a Dimension's Range

By default, the range of the dimension’s axis is set to the rounded minimum and rounded maximum values present in the current dataset.

Options to Set Dimensions Range in the Chart:
Adjust the range of a dimension from its context menu, where three options are available:

  • By clicking Set Range to set the range to specific values.
  • By clicking Set Range from Data to set the dimension to the original range.
  • By clicking Set Rounded Range from Data to reset the dimension to the rounded values.

Filter Records

Records can be activated and deactivated by setting filters on one or more dimensions. Records outside of the range are automatically deactivated and greyed out.

Options to Filter Records in the Chart:

  • A double-edged range slider can be manipulated on every dimension to filter out records by values on that dimension.
  • Set and reset filters from the dimension’s context menu. With Set Filter a filter to specific values can be set and by clicking Reset Filter the filter gets reseted to include all records.

Selecting Records

It is helpful to be able to select one or more records in the dataset and to highlight them accordingly.

Options to Select Records in the Chart:

  • Left-clicking one or more polylines selects the corresponding record(s) and highlights them in orange.
  • Shift-left-clicking adds one or more records to the current selection.
  • Control-left-clicking toggles the selection status of the corresponding records.
  • Select, unselect records from a records context menu.
  • Left-clicking in empty space clears the current selection.

Hovering over Records

If there are a large number of records, a parallel coordinates visualisation contains a large number of potentially crossing and overlapping polylines, so it is challenging to recognise which polyline belongs to which record.

Option to Hover over Records in the Chart:

Hovering over one or more polylines highlights the polyline(s) beneath the mouse pointer in red and displays tooltips with the labels and values of the corresponding records.
