Usage

Parallel coordinates are used for:

  • Discovering clusters and outliers among records.
  • Finding correlations between dimensions.

Identifying Outliers and Clusters of Records

Outliers stand out clearly because their polylines follow a different path to those of other records. Clusters are groups of data records which have similar characteristics, so their polylines follow similar paths.

Looking for Correlations Between Dimensions

If two dimensions are positively correlated, high values in one dimension correspond to high values in the other. Low values in one dimension correspond to low values in the other. In a parallel coordinates plot, this can be seen only when the two dimensions are adjacent to one another. Parallel, non-crossing straight line segments run betwen the two dimensions, as shown in Figure 1a. In a scatterplot, this corresponds to points falling along straight line, as shown in Figure 1b.

If two dimensions are negatively correlated, low values in one correspond to high values in the other and vice versa. In a parallel coordinates plot, the line segments between the adjacent dimensions all cross in the middle, as shown in Figure 3a. In the equivalent scatterplot, the points fall along a negatively inclined straight line, as shown in Figure 3b.
